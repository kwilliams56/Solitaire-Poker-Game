# Solitaire Poker Game

A card-based poker game created in C for my CS100 course.

## About
This project is based on Five Card Draw poker. Players are dealt cards and can replace up to three cards to improve their hand and score points.

## Features
- Card shuffling
- Poker hand evaluation
- Score tracking
- Interactive gameplay

## Technologies Used
- C Programming Language

- Randomization
- Conditional logic
